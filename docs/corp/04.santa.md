# 산타

**2023.04 - 2024.12** · 정규직 · Monorepo·코어 웹·React Native 앱 등 전사 프론트 효율·품질 개선

## Summary


**Lerna 기반 Monorepo**와 **Rollup 빌드**, **SSO·공통 모듈** 정리로 다레포·중복을 줄이고, 코어 서비스·대시보드·**React Native** 앱까지 **폼·상태·폴더 구조·분석 스크립트**를 일관되게 맞춘 경험.

추가로, **어드민·수강생 페이지·디자인 라이브러리** 3축을 Monorepo로 묶어 배포 없이 검증한 사례는 **§1**에 정리함.

## 0. 수강생·운영자 프로젝트 로그인·인증 통합

### Problem

- **수강생**용과 **운영자(어드민)**용 **프로젝트가 각각** 있었고, **같은 DB**를 쓰는데도 로그인·인증이 **동일하게 중복** 구현되어 있었음. 그중에서도 **인증 관련 로직**이 겹쳐 **두 프로젝트를 따로** 유지보수해야 하는 부담이 큼

### Approach

- 중복되던 **인증·로그인 로직을 통합**하고, **이미 운영 중인 서비스**를 안정적으로 유지한 채 **레거시를 제거**하며 한 흐름으로 맞춤
- 통합된 인증을 **Lerna Monorepo**로 **공통 패키지**에 두고, **Rollup** 빌드(**CommonJS / ESM**), **Feature Toggle**, **Circle CI** 자동 테스트, **SSO(단일 로그인)** 중심으로 배포·검증 프로세스를 정리

### Result

- 인증·로그인 **유지보수를 2개에서 1개**로 줄여 **중복 제거** — 레포·패키지 경계도 정리되어 **개발·배포 효율**과 **재사용·유지보수성**이 함께 좋아짐

## 1. 어드민 · 수강생 페이지 · 디자인 라이브러리 (워크스페이스 연동)

### Problem

- **어드민**, **수강생(학습) 페이지**, **공통 디자인 라이브러리**가 각각 있던 구조에서, 라이브러리(디자인 시스템)를 테스트하려면 **배포 → 확인**을 반복해야 했고 **버전 번호만 빠르게 올라가는** 비효율이 큼

### Approach

- **Monorepo**로 세 축을 한 워크스페이스에서 묶어, 라이브러리를 **배포하기 전에** 어드민·수강생 프로젝트에 **바로 연결·적용**해 보며 검증할 수 있게 관리
- **배포 없이** 디자인 시스템 변경을 반복 적용·테스트할 수 있는 흐름으로 전환

### Result

- 배포 없이 라이브러리·디자인 시스템을 빠르게 시험·적용할 수 있어 **개발 효율이 50% 이상** 향상(배포만 반복되던 패턴 완화)

## 2. 코어 서비스·대시보드·코드 구조

### Problem

- **테마 빌더**·**대시보드** 등 코어 기능이 커지면서 폼·상태가 복잡해지고, 레포마다 **폴더 구조**가 달라 온보딩 비용이 큼
- **500줄 이상** 규모의 대형 컴포넌트로 가독성·수정 리스크가 증가

### Approach

- 테마 빌더 관련 코드 개발 및 유지보수
- **React Hook Form + Yup**으로 폼 로직 구현
- 수강생 현황·조회수·결제 내역 등 **대시보드** 기능 개발
- **3개 이상 레포**에 공통 **폴더 구조 표준화**로 컨벤션 통일
- 대형 컴포넌트를 **Context 기반 상태 분리**·책임 단위별 컴포넌트 분해로 리팩터링

### Result

- 신규 기능 진입 장벽 감소, **가독성·유지보수성** 개선

## 3. 분석·배포·모바일 앱

### Problem

- 마케팅·개발용 **스크립트 삽입**이 산재하고, 앱은 **본인인증·영상·실시간 동기화** 등 요구가 다양함

### Approach

- **GA·GTM** 등 스니펫을 **GTM**으로 정리해 삽입
- 개발 배포와 분리된 **CI/CD**로 운영 효율 확보
- **React Native**로 iOS·Android **크로스플랫폼** 기능 개발·개선, 성능·로딩 최적화, OS·디바이스별 이슈 대응

### Result

- 추적·배포 흐름이 정리되고, 앱 **안정성·UX**를 유지하는 기반이 마련됨

## Tech Stack

React, React Native, TypeScript, Lerna, Rollup, Circle CI, React Hook Form, Yup, GA, GTM 등
