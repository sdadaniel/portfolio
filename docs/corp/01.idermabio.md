# 아이더마바이오

**2018.02 - 2019.01** · 정규직 · 웹페이지 제작 및 운영 관리

## Summary

**그누보드** 기반 레거시 사이트 유지보수와 **카페24** 기반 신규 사이트 구축·운영을 병행하며, 콘텐츠·이미지 중심 변경과 신규 채널 오픈을 동시에 맡은 경험.

## 1. 그누보드 기반 홈페이지 유지보수

### 개요

운영 중인 그누보드 기반 회사 홈페이지의 **유지보수** 및 **콘텐츠·이미지** 반영 업무를 담당.

### 결과

- 요청 사항을 반영해 **중단 없이** 일상 운영 리듬 유지

## 2. 카페24 기반 신규 홈페이지

### 개요

**카페24** 환경에서 신규 홈페이지 **구축** 후 **운영 관리** 담당.

### 결과

- 신규 채널을 **출시 후 운영** 가능한 상태로 마무리

## Tech Stack

그누보드, Cafe24, HTML/CSS 등(이력서 기준)
