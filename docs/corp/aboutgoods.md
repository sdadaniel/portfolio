# 어바웃굿즈

**2019.06 - 2021.01** · 정규직 · 카페24 기반 쇼핑몰 제작·운영, 1인 프론트엔드 전담

## Summary

카페24 솔루션 기반으로 다수 쇼핑몰을 단독 구축하고, 자사 브랜드 사이트 유지보수·SEO·분석 체계를 함께 갖춘 1인 개발 환경에서 운영 효율과 데이터 기반 의사결정을 동시에 맞춘 경험.

## 1. 쇼핑몰 구축 및 브랜드 운영

### Problem

- 1인 개발로 신규 브랜드 런칭(약 2달에 1개 수준)과 기존 사이트 유지보수가 병행되어 반복 작업·일정 부담이 큼
- 디자인·기능 요구가 브랜드마다 달라 일관된 생산 방식이 없으면 속도를 내기 어려움

### Approach

- 카페24 기반 쇼핑몰 **8개 이상** 단독 제작(Vanilla JS, jQuery로 UI·기능 구현)
- 자사 브랜드 **12개** 사이트 유지보수·기능 고도화 전담
- 디자인 플로우 등에서 **옵션으로 조정 가능한 템플릿**을 선택해, 단일 고정 레이아웃보다 브랜드별 변형에 대응
- 신규 페이지·이벤트/프로모션·구조 개선까지 운영 전반을 프론트 관점에서 관리

### Result

- 빠른 런칭 리듬에 맞춰 다수 몰을 지속적으로 출시·운영
- 템플릿 전략으로 신규 구축·수정 비용을 줄이고 유지보수 범위를 통제하기 쉬운 형태로 정리

## 2. SEO·웹 분석

### Problem

- 쇼핑몰 수가 늘어나며 페이지별 검색·유입·전환을 잡기 어렵고, 마케팅·운영 판단에 쓸 데이터가 부족함

### Approach

- 페이지별 **메타 태그·robots.txt** 등 SEO 구조 정비
- **GA·GTM** 설치 및 **Data Studio**로 트래픽·전환 분석 리포트 작성
- 유입 경로·콘텐츠 성과를 보고 전략 수립에 필요한 데이터 제공

### Result

- 검색·캠페인 유입을 점검할 수 있는 기준 마련
- 데이터 기반으로 프로모션·페이지 우선순위를 논의할 수 있는 환경 구축

## 3. 운영 자동화(발주·재고 등)

### Problem

- 주문·재고·발주 등 운영 업무가 수작업·반복 작업으로 쌓여 1인 운영 한계를 빠르게 드러냄

### Approach

- 엑셀 기반 자동화(자체 코드·**VBA**)로 발주·재고 현황 등 반복 프로세스를 줄이는 방향으로 정리

### Result

- 반복 입력·집계 시간을 줄여 같은 인력으로 더 많은 브랜드·몰 운영에 집중 가능한 구조로 개선

## 관련 브랜드·채널(정리 메모)

- 쇼핑몰·채널 예: 한삼근, 한삼근 블랙라벨, 핫트랙스, 이지웰, 카페24 등(이력서·내부 자료 기준 보완 가능)

## Tech Stack

JavaScript, jQuery, Cafe24, GA, GTM, Data Studio, Excel/VBA
